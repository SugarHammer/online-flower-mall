#### 软件架构
Java + SpringBoot + Vue + Mysql

#### 项目说明

1.  管理员功能模块
> - 个人中心
> - 用户管理
> - 鲜花分类
> - 鲜花商品信息
> - 订单管理（退款，完成，发货，支付等）
> - 订单评价管理
> - 在线客服
> - 轮播图管理
> - 商品资讯(系统公共)
2.  用户功能模块
> - 首页
> - 商品信息
> - 个人中心
> - 购物车
> - 地址、订单、收藏管理
> - 在线客服咨询
